## Como rodar a aplicação na sua maquina

Clone a aplicação para um repositório no seu computador, entre na pasta raiz da aplicação e execute o comando.


```
yarn
```

ou

```
npm install
```

Depois, inicie o serviço de front-end com o comando:

```
yarn dev
```
ou

```
npm run dev
```
